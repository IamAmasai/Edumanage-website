# EduManage Website

EduManage is a modern, offline-first school management platform designed specifically for African institutions. This repository contains the source code for the landing page.

## Project Structure

- `edumanage-v6.html`: The main landing page (latest version).
- `style.css`: External stylesheet (synced with internal styles in the HTML).
- `script.js`: External JavaScript for scroll animations and form handling.
- `logo.png`: Project logo.
- `edumanage-v4.html`: Legacy version of the landing page.

## Key Features (Landing Page)

- **Responsive Design**: Works across desktop, tablet, and mobile.
- **Scroll Animations**: Smooth reveal animations using Intersection Observer.
- **Waitlist Form**: Simple email validation and submission feedback.
- **Interactive Mockups**: Visual representations of the EduManage dashboard.

## Tech Stack

- HTML5
- Vanilla CSS3 (Custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (ES6+)

## Current Status

The landing page is a high-fidelity prototype with 8 live modules described, a transparent pricing structure, and a "founding school" waitlist.
